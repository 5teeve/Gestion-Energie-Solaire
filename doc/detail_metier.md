# Projet Solaire — Fonctionnalités Attendues

# Objectif général

Créer une application capable de :

- analyser la consommation électrique d’une maison
- dimensionner un système solaire adapté
- estimer les économies réalisées
- afficher des résultats clairs et exploitables

---

# Fonctionnalités principales

## Gestion des sessions

L’utilisateur peut :
- créer une nouvelle session
- enregistrer une étude
- consulter l’historique des études

### Résultat attendu
- chaque simulation est sauvegardée
- historique consultable à tout moment

---

# Gestion des appareils électriques

L’utilisateur peut :
- ajouter un appareil
- modifier un appareil
- supprimer un appareil
- définir :
  - nom
  - puissance
  - heure de début
  - heure de fin

### Résultat attendu
- liste complète des appareils utilisés dans le calcul
- consommation correctement prise en compte

---

# Gestion des tranches horaires

L’utilisateur peut :
- consulter les tranches horaires
- ajouter une tranche
- modifier une tranche
- supprimer une tranche

### Résultat attendu
- système adapte les calculs selon les périodes de la journée

---

# Gestion des rendements énergétiques

L’utilisateur peut :
- configurer les rendements :
  - panneau solaire
  - batterie

### Résultat attendu
- calculs adaptés aux performances réelles du matériel

---

# Gestion des prix électriques

L’utilisateur peut :
- définir le prix du kWh
- modifier les tarifs
- gérer les différences :
  - jour ouvrable
  - weekend

### Résultat attendu
- estimation financière réaliste

---

# Gestion des heures de pointe

L’utilisateur peut :
- définir les heures de pointe
- définir les majorations tarifaires

### Résultat attendu
- calcul précis du coût réseau électrique

---

# Calcul du panneau solaire

Le système doit :
- analyser la consommation
- déterminer la puissance du panneau nécessaire

### Résultat attendu
- puissance panneau recommandée

---

# Calcul de la batterie

Le système doit :
- calculer les besoins énergétiques nocturnes
- intégrer les pertes énergétiques

### Résultat attendu
- capacité batterie recommandée

---

# Calcul du convertisseur

Le système doit :
- détecter les pics de consommation
- appliquer une marge de sécurité

### Résultat attendu
- puissance convertisseur recommandée

---

# Calcul énergétique

Le système doit :
- comparer production et consommation

### Résultat attendu
- surplus énergétique
ou
- déficit énergétique

---

# Calcul financier

Le système doit :
- calculer le coût de consommation réseau
- calculer les économies réalisées grâce au solaire

### Résultat attendu
- estimation gain journalier
- estimation coût évité

---

# Affichage des résultats

Le système doit afficher :
- puissance panneau
- capacité batterie
- puissance convertisseur
- surplus/déficit
- coût ouvrable
- coût weekend

### Résultat attendu
- synthèse complète et lisible

---

# Historique

Le système doit permettre :
- consultation anciennes simulations

### Résultat attendu
- comparaison entre différentes études

---

# Validation des données

Le système doit empêcher :
- valeurs invalides
- données incohérentes
- puissances négatives
- champs obligatoires vides

### Résultat attendu
- données fiables
- erreurs utilisateur réduites

---

# Performance attendue

Le système doit :
- répondre rapidement
- supporter plusieurs appareils
- effectuer les calculs automatiquement

### Résultat attendu
- expérience fluide
- calculs instantanés

---

# Résultat final attendu

L’application doit permettre à un utilisateur de :

- saisir ses appareils électriques
- obtenir automatiquement un dimensionnement solaire
- connaître :
  - le matériel recommandé
  - la consommation estimée
  - les économies possibles
  - les limites du système

# Objectif utilisateur final

Permettre à un utilisateur de :

- comprendre sa consommation électrique
- savoir quel matériel acheter
- estimer ses économies
- optimiser son installation solaire
- prendre une décision basée sur des calculs réels
